# 1.6 As a player, each time I land on or pass “GO” on the game board I want the banker to pay me two hundred dollars.
## User Acceptance test Normal Case for passing or landing on Go
1. The User lands on or goes pass the GO square
2. The system checks if the user has landed on or passed the GO square
3. If the user has the System adds 200 dollars to the players current balance
4. The system displays "Your balance is now " + currentPlayer.getBalance()
